To ensure accessibility for local shop owners (Kirana stores), BazaarSense AI supports voice commands in Hinglish. This allows users to interact with their data without typing.

### Voice Command Examples:
* **Inventory Check:** *"Bhaiya, agle hafte kitna doodh mangwana hai?"*
  * **AI Action:** Analyzes historical demand and local festival trends to suggest order quantity.
* **Market Intelligence:** *"Competitor ne chawal ka price kya rakha hai?"*
  * **AI Action:** Scrapes local marketplace data to provide real-time pricing benchmarks.
* **Profit Summary:** *"Aaj ki bachat kitni hui?"*
  * **AI Action:** Calculates daily margins and net profit instantly.
